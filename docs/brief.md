# Brief — Bulldog Las Vegas

## Qué es

Sistema de 3 agentes de voz AI outbound con identidades distintas para vender servicios de carpet cleaning. Cada agente tiene una personalidad y approach diferente para maximizar conversiones.

## Problema que resuelve

Automatizar las llamadas outbound de ventas de carpet cleaning, eliminando la necesidad de un equipo humano para el primer contacto. El sistema también se mejora solo — analiza sus propias llamadas y ajusta los prompts continuamente.

## Flujo en una línea

Agente llama → Retell envía transcripción → OpenClaw analiza → Human-in-the-Loop aprueba mejoras → prompt actualizado.

## Cliente

Externo — Bulldog Las Vegas

Tiene un agente inbound anterior como base para adaptar a outbound.
